# User Registry CLI

A simple Python CLI project that demonstrates:
- CRUD operations for users
- JSON persistence (save/load)
- Input validation and error handling
- Basic separation of CLI layer and storage logic

## Run

python3 user_registry.py
Commands
add
list
get <id>
delete <id>
update <id>
search <text>
help
exit

Data storage

Data is stored in users.json (created on first save).
